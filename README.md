# Amazon Clone

A responsive **Amazon homepage clone** built using **HTML** and **CSS**. The project replicates the layout and design of Amazon's landing page to practice front-end web development skills.

## Features
- Responsive homepage layout
- Navigation bar
- Hero section
- Product categories
- Footer section

## Technologies Used
- HTML
- CSS

## How to Run
1. Download or clone the repository.
2. Open `index.html` in your web browser.
